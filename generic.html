Server utilities

The following projects were entirely made by me and combined into one single server. If you wish to use any of these, contact me.

King of the Hill

Select an area. The player which stays on the selected area for the longest wins. Rewards and area setting customizable from in game. Comes with a timer and a points counter (Scoreboard intergration)

Alt detection

Detect player alts. Get notified when an alt joins. Able to detect offline players too.

Pets

Pets that follow the player which grants custom perks and abilities. Levels go from 1 - 100. Each level grants more power in certain pre-set fields.

Combat Log

Players get combat-logged upon damaging one another. While combat-logged, the player may not execute commands. If a player leaves while in combat, they die.

Custom Enchanting Table

Set prices, maximum level, and the description of enchants. This system allows to add custom enchantments without any issues.

Punishment System

Give OPs access to (temp)ban(ip), (temp)mute(ip), kick, warn, clear chat and more. Saves all punsihment on an individual GUI for each player.

Custom Crates

Create, Modify and delete crates in game. Select the rewards, the key item and the crate block. See all crates as a list on an organized GUI.

Regenerating Mines

Create mines with random blocks. Select a time at which all mines reset in-game. See all mines as a list on an organized GUI.

Discord Implementation

Log chat, commands, join and leave, and more. Set IDs and tokens by an in-game command. Adds commands such as /players and /stats. Interact with the minecraft chat via the discord chat.

Vaults

Watch, view, and modify vaults. Assign vaults by permissions.
